# Springboot Vue课程管理系统 java298

## 项目概述


## 技术栈

- **后端：** SpringBoot、Mybatis-Plus
- **前端：** Vue
- **数据库：** MySQL

## 功能模块

### 系统角色

- 教务管理员
- 教师
- 学生

不同角色登录后，将看到不同的功能菜单。

### 功能介绍

#### 教务管理员角色

- 管理员首页
- 系管理
- 专业管理
- 班级管理
- 学生管理
- 教师管理
- 课程管理
- 选课修改
- 管理员管理

#### 教师角色

- 教师首页
- 授课查询
- 教师课表
- 成绩录入

#### 学生角色

- 学生首页
- 选修课程
- 学生课程
- 课表查询
- 成绩查询
- 信息维护

## 运行环境

- **后端环境：** 
  - JDK 1.8 或以上
  - Maven 3.6 或以上
  - MySQL 5.7 或以上
  
- **前端环境：**
  - Node.js 12 或以上
  - npm 或 yarn

## 部署步骤

1. 数据库环境搭建及脚本执行
2. 后端服务编译与部署
3. 前端项目编译与部署

具体步骤请参考详细文档及视频教程。
